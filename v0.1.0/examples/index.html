Examples

Run examples from the repository root:

julia --project=examples -e 'using Pkg; Pkg.instantiate()'
julia --project=examples examples/01_minimal_app.jl
FileFocus
01_minimal_app.jlSmall animated app
02_live_reload.jlRevise workflow
03_orbital_dynamics.jlInteractive four-body simulation
04_scene_viewer_3d.jlMeshes, textures, lighting, and camera controls
05_rendering_gallery_2d.jl2D drawing and offscreen canvases

Each file defines a module with main() and also runs as a script.

Interactive example runs are opt-in:

MIRAGE_TEST_INTERACTIVE=1 julia --project -e 'using Pkg; Pkg.test()'

Select examples with MIRAGE_TEST_EXAMPLES=orbital_dynamics,scene_viewer_3d.
